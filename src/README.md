# ToDo console App

Simple Java console application for managing tasks.

-----
## Features 
- Add tasks 
- Mark tasks as completed
- Delete tasks
- Show completed tasks
- Delete all completed tasks
- Show pending tasks
- Automatic task saving after every change

------
## Technologies
- Java
- OOP
- File I/O
- ArrayList

-------
## Development Process 
The application was built as a console-based task manager to practice core Java concepts.

The project was developed step-by-step:

1. Implemented a basic console menu system using `Scanner`.
2. Created a `Task` class to represent tasks using object-oriented programming principles.
3. Stored tasks in an `ArrayList` for dynamic task management.
4. Implemented functionality for adding, completing, and deleting tasks.
5. Added task filtering (completed and pending).
6. Implemented file persistence using `BufferedReader` and `BufferedWriter`.
7. Added automatic saving of tasks after every change.
